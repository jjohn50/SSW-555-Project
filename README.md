```console
pip3 install PrettyTable
```

install PrettyTable (for Windows): 
```console
pip install PrettyTable
```

To run Gedcom parser:
```console
python3 -u parser.py Zaccaria.ged
```

```console
for i in test_*; do echo $i; python3 -u $i; echo; echo; done
```
