To run Gedcom parser:
```console
python3 parse_gedcom Zaccaria.ged
```

To run one test case:
```console
python3 test_USxx_your_description_here.py
```

To run all test cases:
```console
for i in test_*; do echo $i; python3 $i; echo; echo; done
```
